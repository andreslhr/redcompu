Windows 7 es el sistema operativo que ha remplazado a Windows Vista con nuevas utilidades y reutilizando herramientas pasadas para una mejor sincronización con el usuario. Después de las críticas a su antecesor, Microsoft nos brinda un sistema operativo que recoge lo mejor de Windows Vista y añade nuevas características para un funcionamiento más ligero y sencillo.
Requisitos para su instalacion: Si deseas instalar Windows 7 debes tener en cuenta una serie de requisitos mínimos para su adecuado funcionamiento. Primero, necesitas un procesador de 1 GHz de 32 bits o de 64 bits. Segundo, debes contar con 1 GB de RAM o 2 GB de RAM y 16 GB de espacio disponible en tu disco duro. Por último, tener una tarjeta gráfica DirectX 9 con controlador WDDM 1.0.
¿Que tipo de Arquitectura elegir? Recuerda que Windows 7 de 32 bits es para computadores que tienen memoria RAM de 1 Gb, Si tienes más de 2 Gb en tu memoria RAM te puede servir el Windows 7 de 64 bits.
Recuerda que el soporte de Windows 7 finalizó el 14 de enero de 2020
